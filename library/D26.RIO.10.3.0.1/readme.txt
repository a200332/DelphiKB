December 2018 Missing Files Patch for 10.3
This patch resolves issues of missing PCRE obj files if you do not install DUnitX and missing java source files for some of the new enhancements.

Please follow the installation steps below:

1. Unzip the downloaded patch to a temporary directory
2. Copy Java.sources.zip and pcre16.win32.objs.zip to <install-root>* directory
3. Expand these zip files
4. Delete Java.sources.zip and pcre16.win32.objs.zip

* <install-root> is the main root folder of your RAD Studio installation, i.e. C:\Program Files (x86)\Embarcadero\Studio\20.0\
