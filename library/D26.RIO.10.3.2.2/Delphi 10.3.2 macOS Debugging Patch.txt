Delphi 10.3.2 macOS Debugging Patch
================================================

This patch addresses the following issues:

* When debugging a 64-bit macOS Delphi application, some variables passed as parameters would not show the correct data when evaluated or inspected

Installation instructions
-------------------------

1. Shut down RAD Studio
2. Create a backup copy of the following files in your RAD Studio installation folder:
   1. bin\dccosx64.exe, bin\dccosx64260.bpl, bin\dccosx64260.jdbg
   2. The entire lib\osx64 folder
3. Extract the ZIP archive
4. Copy the contents of the ZIP archive to the RAD Studio installation folder, so that the contents overwrite the files in \bin and \lib\osx64
5. Restart RAD Studio
